### Added

- European-first sovereignty strategy covering EU/EEA governance, data residency, provider exit,
  controlled model improvement, and model-serving independence.
- Architecture decision and product gates for European compute pilots and production continuity.
- Initial repository scaffold.
- Phase 1 interest registration with qualification fields for compute, software, research, and
  design partners.
- NVIDIA/CUDA, AMD/ROCm, Apple Silicon, operating-system, availability, and network capability
  collection.
- Server-side validation, hashed verification and deletion tokens, abuse metadata, and generic
  anti-enumeration responses.
- Supabase schema with RLS, least-privilege grants, consent evidence, and retention timestamps.
- Email verification, non-persistent local preview mode, privacy notice, and self-service deletion.
- Reproducible local Supabase stack, database scripts, pgTAP security tests, retention cleanup, and
  database-backed local verification mode.
- Protected operator review queue with signed HttpOnly sessions, same-origin mutation checks,
  qualification filters, atomic decisions, and an append-only audit log.
