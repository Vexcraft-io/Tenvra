# Operations Runbook

## Current Stage

The repository is a scaffold. No production operations exist.

## Future Operational Minimum

- Public status page.
- Structured logs, metrics, and traces without prompt content.
- On-call ownership and escalation contacts.
- Backup restore exercises.
- Certificate, signing-key, and dependency-expiry monitoring.
- Capacity, cost, and contributor-pool alerts.
- Documented deploy, rollback, model promotion, and job-pause procedures.

Production access requires phishing-resistant MFA, least privilege, just-in-time elevation, and
auditable actions.
